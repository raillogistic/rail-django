Quickstart

  1. Define models inside the apps/ directory.
  2. Run migrations to build tables.
  3. Query the schema from the GraphQL endpoint.
python manage.py makemigrations
python manage.py migrate
python manage.py runserver

Key endpoints

  • POST
    /graphql/gql/ Main protected API endpoint
  • POST
    /graphql/auth/ Public auth mutations only
  • GET
    /schemas/ Schema registry overview
  • GET
    /graphql/graphiql/ Interactive IDE when enabled
