# Contributing to Rail Django

Thank you for your interest in contributing to Rail Django!

## Getting Started

1.  **Fork** the repository on GitHub.
2.  **Clone** your fork locally.
3.  **Install** dependencies:
    ```bash
    pip install -e .[dev]
    ```

## Development Workflow

1.  Create a new branch for your feature or fix:
    ```bash
    git checkout -b feature/my-feature
    ```
2.  Write code and add tests.
3.  Run tests to ensure everything is working:
    ```bash
    pytest
    ```

## Code Style

*   We use `black` for formatting.
*   We use `isort` for import sorting.
*   We use `mypy` for static type checking.

Please ensure your code passes these checks before submitting.

## Pull Requests

1.  Push your branch to GitHub.
2.  Open a Pull Request against the `main` branch.
3.  Describe your changes and why they are needed.
